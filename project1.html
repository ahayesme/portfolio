Pacman

Modeled after the original Pacman game, I have coded a version of the video arcade game using Java.

Figure 1: Pacman game, just begun.

Pacman is played on a 23 x 23 grid organized into a maze. The player controls Pacman’s direction of movement with the keyboard (the right, left, up and down arrows navigate Pacman in the corresponding direction through the maze). The goal is to eat all of the white dots populating the maze while avoiding the four ghosts (Blinky, Pinky, Inky, and Clyde) who chase after Pacman.

The user begins with three lives and loses one each time Pacman is caught by a ghost. When Pacman no longer has remaining lives, the game ends and the player has lost. The player’s score is calculated throughout the game (10 points are earned for each white dot eaten, 100 points for the energizers, and 200 points for each ghost eaten during the period after an energizer has been eaten and the ghosts are consumable). If Pacman eats all the white dots in the maze before losing all 3 lives, the player has won! The game can be paused at any point in play by using the “P” key and game be restarted from the beginning by hitting the restart button.

Tetris

Originally programmed, designed, and released in the 80s, Tetris is a puzzle video game that challenges the user to match tiles. The following project is a recreation of this popular game, which I coded in Java during a course which introduced object-oriented design in Java.

My game is modeled much like a standard Tetris game: When the user launches the game, an empty game board appears and one of seven randomly selected Tetris puzzle pieces begins to fall down the board. When a piece can no longer move (i.e. it has reached the bottom of the screen or there is a Tetris piece already occupying the space below it), the currently falling piece reaches its final resting location and a new piece begins to fall.

The player can control the location of the currently falling piece at any time by using the right and left keys for horizontal movement, the down arrow to move the piece one square downward, the space bar to drop the piece to the lowest possible resting point, or the up arrow to rotate the current piece counterclockwise. A player may also pause the game at any point using the “P” key.

The objective of the game is to fill complete rows of the game board with Tetris tiles. When a row is completed, it will disappear and all rows above it will be shifted downwards. Game play ends if Tetris pieces get stacked to the top of the board and there is no longer room for new pieces to be generated.
